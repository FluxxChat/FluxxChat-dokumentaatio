## Contributing CSS

CSS files of the project are in SCSS format. Every file that uses styling has it's own SCSS file in the same directory.

![Example 1](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-1.png)

Every SCSS files import colors.scss file from the project root.

![Example 2](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-2.png)

If you want to add colors to the project, first add them to colors.scss, then use them according to examaple of the image on the bottom.

![Example 3](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-3.png)

![Example 4](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-4.png)
