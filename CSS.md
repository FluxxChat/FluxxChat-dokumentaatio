## Osallistuminen CSS

Projektin tyylitiedosto on SCSS muotoa. Jokaiselle tyylitystä käyttävälle tiedostolle on oma SCSS tiedosto samassa hakemistossa.

![Esimerkki 1](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-1.png)

Tyylitiedostot perivät projektin juuressa olevan colors.scss tiedoston.

![Esimerkki 2](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-5.png)

Tyylit otetaan .ts tiedostoissa käyttöön alla olevan kuvan mukaisesti.

![Esimerkki 3](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-2.png)

Jos haluat lisätä projektiin värejä lisää ne ensin uutena värinä colors.scss tiedostoon, ja käytä sitä alimman kuvan esimerkin mukaisesti.

![Esimerkki 4](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-3.png)

![Esimerkki 5](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-4.png)

## Contributing CSS

CSS files of the project are in SCSS format. Every file that uses styling has it's own SCSS file in the same directory.

![Example 1](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-1.png)

Every SCSS files import colors.scss file from the project root.

![Example 2](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-5.png)

Styles are implemented in .ts files according to example of the picture below.

![Example 3](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-2.png)

If you want to add colors to the project, first add them to colors.scss, then use them according to examaple of the image on the bottom.

![Example 4](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-3.png)

![Example 5](https://github.com/FluxxChat/FluxxChat-dokumentaatio/blob/master/img/css-4.png)
